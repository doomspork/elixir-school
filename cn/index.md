# 关于 Elixir
"Elixir 是一种动态的函数编程语言，用来编写可扩展和易于维护的应用。" --[elixir-lang.org](http://elixir-lang.org/)

Elixir 语言是构建在久经检验的 Erlang VM (BEAM) 之上，可以方便地编写分布式、低延迟和高容错性的系统。

## 特性
- 高并发
- 高容错性
- 函数式编程
- 可扩展
